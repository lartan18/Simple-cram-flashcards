What is DAC?\**Discretionary Access Control (DAC)** allows the resource owner to decide who has permissions to access their files or objects.$
What is MAC?\**Mandatory Access Control (MAC)** is a model where security policies are enforced by a central authority regardless of individual user preferences.$
What is the Biba Model?\The Biba model is a MAC model focused on integrity, enforcing "no write up" and "no read down" rules.$
What is the difference between RBAC and ABAC?\**RBAC** assigns access based on a user's organizational role, while **ABAC** provides fine-grained control based on attributes like time, location, or department.$
What is Single Sign-On (SSO)?\SSO allows a user to log in once and gain access to multiple independent applications without re-authenticating for each one.$
What is the role of an Identity Provider in SSO?\The Identity Provider (e.g., Feide) is responsible for **authenticating the user** and sending confirmation to the Service Provider (e.g., Blackboard).$
What is the difference between OpenID Connect and OAuth 2.0?\**OpenID Connect** is used for authentication (verifying who a user is), while OAuth 2.0 is used for authorization (granting access to specific data or APIs).$
What is a Buffer Overflow attack?\It is an attack where a buffer is flooded with more data than it can hold, causing the excess to overflow into adjacent memory, potentially leading to **malicious code execution**.$
What is an Access Control List (ACL)?\An Access Control List is a list attached to a resource that specifies which users or groups have permissions to access it and what actions they can perform.$
What is a Capability?\A capability is an access control model where permissions are stored **with the user** rather than with the object, allowing portable delegation of access rights.$
What is SAML 2.0?\**SAML 2.0** is an **XML-based standard** used for SSO that enables secure authentication and authorization requests and responses between identity providers and service providers.$
What is a Type-Safe Language?\A type-safe language (like Java or Python) is one that **automatically manages memory allocation**, preventing buffer overflows and similar memory-based vulnerabilities that can occur in languages like C.$
