What is the difference between a Principal and an Identity in access control?\A principal is any entity (user, process, or device) capable of performing actions in a system, while an identity is the unique label or identifier used to distinguish that principal from others. $ Explain the access control concepts of DAC and MAC?\DAC (Discretionary Access Control) allows resource owners to grant access at their discretion. MAC (Mandatory Access Control) enforces system-wide policies based on fixed security levels (clearances) that users cannot override. $ What are RBAC and ABAC?\RBAC (Role-Based Access Control) grants permissions based on a user's assigned role. ABAC (Attribute-Based Access Control) grants access based on attributes of the user, the resource, and the environment. $ What are the two core properties of the Bell-LaPadula model?\It focuses on confidentiality using the Simple Security Property ("no read up" to higher levels) and the * (Star) Property ("no write down" to lower levels). $ What is the "Strong *" property in Bell-LaPadula?\The Strong * property states that a principal can only read and write at the exact same security level, preventing both unauthorized access to secrets and accidental leakage to lower levels. $ How does the Biba model differ from the Bell-LaPadula model?\While Bell-LaPadula enforces confidentiality, Biba enforces integrity using rules like "no read down" (from low-integrity data) and "no write up" (to high-integrity data) to prevent corruption. $ What is the basic protocol flow of Single Sign-On (SSO)?\The user attempts to access a Service Provider, is redirected to an Identity Provider to authenticate (e.g., via a login screen), and once verified, is redirected back to the Service Provider with a secure token. $ What is SAML 2.0?\SAML (Security Assertion Markup Language) is an XML-based standard used in SSO systems to exchange authentication and authorization data between an Identity Provider and a Service Provider. $ What are OAuth 2.0 and OpenID Connect?\OAuth 2.0 is a framework used for authorization (granting access to resources), while OpenID Connect is an identity layer built on top of OAuth 2.0 specifically for authentication (verifying identity). $ What is LDAP?\LDAP (Lightweight Directory Access Protocol) is a protocol used to manage and access directory information services, often serving as the backend to store user credentials in SSO systems. $ What is a Buffer Overflow attack?\It is a control hijacking attack where a program writes more data to a memory buffer than it is designed to hold, causing the data to overwrite adjacent memory (like return addresses) to change execution flow. $ What types of inputs can trigger a buffer overflow?\Any unsanitized input can cause an overflow, including strings from a user, data read from a file, or values stored in environment variables. $ What is the purpose of a circuit breaker pattern in microservice security?\It is used to prevent a failure in one microservice from cascading and taking down other services, maintaining system availability by stopping requests to a service that is struggling or under attack.
