What is the Vigenère Cipher?\The Vigenère cipher is an encryption method that uses a keyword to shift each letter of the plaintext by a different amount based on the key's letters.$
What is a One-Time Pad (OTP)?\An OTP is a **perfectly secure cipher** if the key is random, at least as long as the message, used only once, and kept secret.$
What is the difference between a Stream Cipher and a Block Cipher?\A stream cipher encrypts data continuously using a short seed, while a block cipher (like AES) encrypts fixed-size blocks of data using a smaller key.$
What is ECB mode and why is it insecure?\**Electronic Code Book (ECB)** mode is insecure because identical plaintext blocks result in identical ciphertext blocks, potentially revealing patterns in the data.$
What is HMAC?\HMAC stands for Hash-based Message Authentication Code; it uses a hash function and a shared secret key to provide both integrity and authenticity for a message.$
How does Public Key Cryptography work?\It uses a **key pair**: a **Public Key** to encrypt messages (which can be shared with anyone) and a **Private Key** to decrypt them (which only the owner knows).$
What is a Certificate Authority (CA)?\A CA is a trusted third party that verifies and certifies that a specific public key belongs to the person or entity claiming it.$
What is Quantum Key Distribution?\It is a future communication method that uses **photon polarization** to establish encryption keys, allowing parties to detect if an eavesdropper is listening.$
What is a Shift Cipher?\A shift cipher is a basic encryption method where each letter of the alphabet is shifted a fixed number of positions (K places) to create ciphertext.$
What is Letter Frequency Analysis?\Letter frequency analysis is a cryptanalysis technique used to break ciphers by examining the frequency of letters in the ciphertext and comparing them to known language patterns.$
What is CBC Mode?\**Cipher Block Chaining (CBC)** mode is a block cipher mode that uses a random **initialization vector** to ensure identical plaintext blocks encrypt to different ciphertext blocks.$
What is a Digital Signature?\A digital signature is created by signing a **hashed message** with a private key, allowing others to verify the author's identity and message authenticity using the public key.$
What is the TLS 1.2 Handshake?\The TLS 1.2 handshake is the **multi-step process** between a browser and server to establish a secure encrypted connection, involving key exchange and cipher negotiation.$
